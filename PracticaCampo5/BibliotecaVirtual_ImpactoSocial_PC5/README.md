# Biblioteca Virtual con Impacto Social (PC5)

Este proyecto consiste en una aplicación de escritorio en Java para la gestión de libros, préstamos y reservas en una biblioteca universitaria.  
Se desarrolla como parte de la práctica de campo 5 del curso **Técnicas de Programación Orientada a Objetos**, incorporando un enfoque de impacto social.  

## Características principales
- Registro de usuarios (estudiantes y docentes).  
- Gestión de libros y préstamos.  
- Reservas y devoluciones.  
- Reportes exportables en CSV.  
- Reflexión sobre el impacto social del software.
